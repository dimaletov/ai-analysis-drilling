# Сравнительный анализ результатов

### Кейс №1 (Техника)
Gemini точнее интерпретировал физические симптомы. Падение давления при неизменном весе — классический признак нарушения герметичности (промыва), что и было указано ботом. YandexGPT предложил версию поглощения, что менее вероятно при данных вводных.

### Кейс №2 (Безопасность)
YandexGPT показал отличный результат в области документального сопровождения работ, что критически важно для инженера в России. Gemini же выдал более детализированный список проверки самого «железа» (талевой системы и ключей).

### Итог
Для инженерных расчетов и поиска неисправностей предпочтительнее Gemini. Для подготовки отчетности и чек-листов по безопасности — YandexGPT.
