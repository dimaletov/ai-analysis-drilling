# Исследование возможностей чат-ботов с ИИ в добывающей промышленности
## Автор: Кузнецов Д. В., группа БНГА-о24 (ЮРГПУ (НПИ))

## Цель исследования
Сравнить качество решения прикладных задач нефтегазовой отрасли с помощью зарубежного (Gemini) и отечественного (YandexGPT) ИИ-ассистентов.

## Используемые модели
1. **Зарубежный бот:** Gemini 1.5 Flash
2. **Отечественный бот:** YandexGPT

## Выбранные кейсы
1. **Кейс 1: Диагностика осложнений при бурении.**
   * Сценарий: Моделирование ситуации с падением давления на манифольде при стабильном весе.
2. **Кейс 2: Промышленная безопасность.**
   * Описание: Формирование чек-листа перед началом спуска обсадной колонны.

## Методология
1. Сформулирован единый запрос для каждого кейса.
2. Результаты сохранены в папку `results/`.
3. Проведен анализ по критериям технической точности и соответствия регламентам РФ.

## Основные выводы
* **Gemini** продемонстрировал глубокое понимание физики процессов, точно определив «промыв» бурильной колонны.
* **YandexGPT** лучше ориентируется в административных вопросах охраны труда и специфике работы в РФ (учет нарядов-допусков и региональных факторов).
